💰 Backend Ledger (Finance / Transaction API)

A backend-based ledger system built using Node.js and Express that allows users to manage financial transactions, track balances, and maintain a structured record of credits and debits.

🚀 Features

🔐 User Authentication (Signup & Login)

💳 Add Income & Expenses

📊 Track Transaction History

💰 Real-time Balance Calculation

📁 Structured Ledger Management

⚡ RESTful API Design

🔒 Secure Backend Handling

🛠️ Tech Stack

Node.js

Express.js

MongoDB

Mongoose

JWT Authentication

Postman
